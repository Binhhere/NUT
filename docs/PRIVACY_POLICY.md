# NUT Privacy Policy

Last updated: 2026-06-21

## Summary

NUT is a private, local-first streak tracker. The v1 app does not require an account, does not use a backend, does not sell data, and does not include advertising or third-party analytics.

## Data Stored On Device

NUT stores the following information locally on the user's device:

- Streak start date, check-in status, best streak, reset count, and lifetime clean days.
- Optional username and optional personal reason chosen during onboarding.
- Optional reminder settings.
- Optional app lock preference.
- Optional backup/import data when the user chooses to export or restore a backup file.

## Data Collection And Sharing

For the current v1 local-first build:

- NUT does not transmit streak, profile, reminder, or reset data to NUT servers.
- NUT does not sell user data.
- NUT does not use advertising SDKs.
- NUT does not use third-party analytics SDKs.
- Mock feed posts are local demo content unless a future moderated backend is added.

## Permissions

NUT may request:

- Notifications, only if the user enables daily reminders.
- Biometrics or device lock, only if the user enables App Lock.
- File access/share flows, only when the user exports or imports a backup.

## User Choices

Users can:

- Skip optional onboarding fields.
- Disable reminders.
- Disable App Lock.
- Export a local backup.
- Import a backup.
- Delete the app to remove local app data from the device.

## Sensitive Subject Matter

NUT is a self-tracking tool for focus and discipline. It is not medical care, therapy, diagnosis, or crisis support. Users should seek qualified professional help for medical, mental-health, or safety concerns.

## Future Changes

If NUT later adds accounts, cloud sync, a public community feed, moderation, analytics, payments, or crash reporting, this policy and the App Store / Google Play privacy disclosures must be updated before release.
